# Uptime Kuma Tray Agent v1.0.6

## Highlights

- Mehrsprachige Oberfläche mit Systemstandard: Deutsch, English und Polski.
- Windows-Sprache wird automatisch genutzt, wenn `language` auf `System` steht.
- MSI-Build erstellt jetzt x64- und x86-Pakete.
- Bestehende Konfigurationen bleiben bei Updates erhalten und werden aus alten AppData-Pfaden migriert.
- GitHub Pages ist über den Ordner `docs` vorbereitet.

## Installer

- `UptimeKumaTrayAgent-Setup-1.0.6-x64.msi` für normale 64-bit Windows-Server.
- `UptimeKumaTrayAgent-Setup-1.0.6-x86.msi` für 32-bit Windows-Systeme.
- `UptimeKumaTrayAgent-Setup-1.0.6-x32.msi` ist nur ein zusätzlich benannter Alias des x86-Pakets.

## Update-Hinweis

Der MSI-Installer aktualisiert bestehende Installationen über das gleiche UpgradeCode-Feld. Programmdateien und Dienst werden aktualisiert, Konfiguration und Logs unter `%ProgramData%\UptimeKumaTrayAgent` bleiben erhalten.
