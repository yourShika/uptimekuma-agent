# Uptime Kuma Tray Agent v1.0.7

## Highlights

- Mehrsprachigkeit erweitert: Tabellen, Tabs, Dialoge, Auswahlfelder, Statuswerte und Dienstaktionsmeldungen werden in Deutsch, English und Polski angezeigt.
- Spracheinstellung bleibt standardmäßig `System` und nutzt damit die Windows-Anzeigesprache.
- Nach einer Sprachänderung zeigt die App eine kurze Meldung, dass ein Neustart nötig ist, damit wirklich alle UI-Texte vollständig wechseln.
- Technische Config-Werte wie `Running`, `Light` oder TCP-Richtungen bleiben kompatibel, werden aber lokalisiert angezeigt.

## Installer

- `UptimeKumaTrayAgent-Setup-1.0.7-x64.msi` für normale 64-bit Windows-Server.
- `UptimeKumaTrayAgent-Setup-1.0.7-x86.msi` für 32-bit Windows-Systeme.
- `UptimeKumaTrayAgent-Setup-1.0.7-x32.msi` ist nur ein zusätzlich benannter Alias des x86-Pakets.
